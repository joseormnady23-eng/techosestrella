# Iconos de la PWA

Faltan los iconos reales para que la app se instale con su ícono propio. Agrega aquí:

- `icon-192.png` (192×192)
- `icon-512.png` (512×512, también se usa como maskable)

Sugerencia: exportar la estrella de Techos Estrella sobre fondo `#1A1A1A` o `#1E7FC2`.
Sin estos archivos la app funciona igual; solo la instalación usará un ícono genérico.
